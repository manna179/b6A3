#  Vehicle Rental Management System (SQL Project)

A PostgreSQL-based database project for managing users, vehicles, and bookings in a vehicle rental system.  
This project demonstrates relational database design and commonly used SQL queries.

---

##  Features

- Structured relational database schema
- Proper use of **Primary Key** and **Foreign Key**
- Data validation using **CHECK** and **UNIQUE** constraints
- Real-world SQL queries using JOIN, WHERE, HAVING, and NOT EXISTS

---

##  Tech Stack

- **Database:** PostgreSQL
- **Language:** SQL

---

##  Database Tables

### Users
- Stores admin and customer information

### Vehicles
- Stores rental vehicle details (car, bike, truck)

### Bookings
- Manages vehicle bookings by users
- Maintains relationships using foreign keys

---

##  Relationships

- One user can have multiple bookings
- One vehicle can have multiple bookings
- Foreign keys ensure data integrity
